# Teste Unitário em Python

Atividade prática de testes unitários usando `unittest` - disciplina de Engenharia de Software (GCC188) na UFLA.

## Pré-requisitos

- Python 3.x
- VS Code (opcional)

## Como executar os testes

```bash
python -m unittest discover
```
